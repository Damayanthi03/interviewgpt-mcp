# InterviewGPT

![InterviewGPT](assets/landingpg.png)

An AI-powered interview preparation platform built using the Model Context Protocol (MCP) to help users prepare for technical and HR interviews.

![Python](https://img.shields.io/badge/Python-3.11-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-Backend-green)
![React](https://img.shields.io/badge/React-Frontend-61DAFB)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-Database-blue)
![MCP](https://img.shields.io/badge/MCP-AI%20Architecture-purple)

## Features

- Resume Analysis
- HR Interview Practice
- Coding Practice
- Aptitude Test
- Communication Skills
- Job Recommendations
- User Authentication

## Tech Stack

- React
- TypeScript
- FastAPI
- PostgreSQL (Neon)
- MCP
- Generative AI

## Setup

```bash
# Clone the repository
git clone https://github.com/your-username/interviewgpt-mcp.git

# Install dependencies
pnpm install

# Start the application
pnpm dev
```
